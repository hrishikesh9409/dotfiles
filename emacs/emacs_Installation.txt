Install dependencies first:

sudo apt-get install -y stow build-essential libx11-dev xaw3dg-dev libjpeg-dev libpng-dev libgif-dev libtiff-dev libncurses5-dev libxft-dev librsvg2-dev libmagickcore-dev libmagick++-dev libxml2-dev libgpm-dev libotf-dev libm17n-dev

Download emacs 24.5:
https://ftp.gnu.org/gnu/emacs/emacs-24.5.tar.gz

Execute the following commands:
./configure CFLAGS=-no-pie
make
make install
